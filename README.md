## 🧩 What this is

nextjs-posthook-ai-review is a small demo app that shows how a human can review AI-generated content before it moves on. It uses Next.js for the app, OpenAI for the AI step, and Posthook for delayed jobs and reminders.

This app is useful if you want to:

- Test an AI review flow
- See how delayed jobs work
- Try a simple web app on Windows
- Learn how webhooks and reminders can fit into one flow

## ✉️ What the demo does

This demo shows a simple path:

- A user submits a review item
- The app sends it to an AI step
- A person checks the result
- Posthook handles a delay or reminder step
- The flow continues after review

This setup helps you see how AI and human review can work together in one process.

## 🛠️ Main parts

- **Next.js**: runs the web app
- **OpenAI**: creates the AI review output
- **Posthook**: manages delayed jobs, reminders, and webhook flow
- **Webhooks**: move data between steps
- **Serverless flow**: helps the app run in a cloud-style setup

## 📌 What you can expect

This app is made for learning and testing. It is not a full business system. It gives you a clear view of:

- How data moves through each step
- How a review queue can pause work
- How reminders help keep tasks moving
- How a simple AI workflow can include a person before final action
